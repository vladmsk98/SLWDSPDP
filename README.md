# 🎵 SLWDSPDP - Аудиоплеер

Простой аудиоплеер с плейлистом.

## 🚀 Запуск

Просто откройте сайт по ссылке

## 🎮 Управление

### Клавиши
- ▶️/⏸️ — Play/Pause
- ⏮/⏭ — Предыдущий/Следующий
- Клик по треку — Переключить
- Прогресс-бар — Перемотка
- Ползунок — Громкость

## 🎨 Особенности

- Тёмная тема
- Плейлист
- Анимированная иконка
- Прогресс-бар
- Регулировка громкости
- Повтор трека
- Сохранение позиции
- Адаптивность

📱 Поддержка

Все современные браузеры
