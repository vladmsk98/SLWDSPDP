# 🎵 SLWDSPDP - Аудиоплеер

Простой и стильный аудиоплеер с анимированной иконкой.

## 🚀 Запуск

Просто откройте сайт по ссылке https://vladmsk98.github.io/SLWDSPDP/

## 🎮 Управление

### Клавиши
- `Space` — Play/Pause
- `→` — +5 секунд
- `←` — -5 секунд
- `M` — Вкл/Выкл звук

## 🎨 Особенности

- Тёмная тема с золотыми акцентами
- Анимированная иконка при проигрывании
- Свечение обложки во время музыки
- Сохранение позиции воспроизведения
- Адаптация под любые экраны
